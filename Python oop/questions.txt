# 1. Student Class (Class + Instance Variables)

Create a class `Student`:

* instance variables:

  * `name`
  * `marks`
* class variable:

  * `school_name`
* method:

  * `show_details()`

Create 2 students and show:

* how instance variables change per object
* how class variables stay shared

Concepts:

* class variables
* constructors
* objects

# 2. Bank Account with Deposit/Withdraw

Create a class `BankAccount`:

Features:

* `deposit(amount)`
* `withdraw(amount)`
* `show_balance()`

Rules:

* balance cannot go below 0
* print proper messages

Concepts:

* encapsulation mindset
* methods
* condition checking

# 3. Inheritance Practice — Employee System

# 5. Polymorphism — Animals

# 6. Duck Typing Challenge

Create:

* `PDFFile`
* `ImageFile`
* `VideoFile`

Each should have:

```python
open()
```

Now create:

```python
def open_file(file):
    file.open()
```

Pass different objects into the function.

Concepts:

* duck typing
* flexible design

# Important Rule

While solving:

* first think on paper
* identify:

  * which object owns what
  * which class should contain which method
  * where inheritance is actually useful
* THEN code

That thinking process is what separates:

* “I know syntax”
  from
* “I can design software”
